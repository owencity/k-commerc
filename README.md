## 6. 구현기능 


###  🍊 김동규
- **UI**
  - 메인페이지 구현
  - 회원 관련 UI (회원가입, 회원수정, 로그인, 로그아웃, 회원탈퇴)
  - 상품 등록 관련 
- **구현기능**
  - 유효성 검사를 활용한 회원가입(bCrypt) 
  - 상품 읽기,등록,수정,삭제
  - JWT 활용한 로그인,로그아웃 구현
  - webSocket을 이용한 실시간 채팅 구현
  - AWS + nginx 배포
